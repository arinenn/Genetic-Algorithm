Генетический алгоритм поиска локального экстремума на полуинтервале от заданной функции.
В данном случае функция имеет вид f(x) = x[(x − 1.1)^5][(x − 1.2)^4][(x − 1.3)^3]cos(x + 100) 
Функция проверяется на полуинтервале [0,4)

1) Необходимым требованием является наличие в рабочей директории файла 'config.txt'
2) Программа выводит все ошибки, связанные с обработкой данных из файла.
3) При запуске программы достаточно следовать инструкциям, чтобы прийти к удачному запуску.
4) Удачный запуск означает получение конечного результата с уведомлением о соответствующем критерии останова.
5) Программа поддерживает работу в двух режимах: тестовый и основной.
    а) Основной режим выводит в консоль конечную абсциссу точки и значение функции в ней.
    б) Тестовый режим выводит популяцию на каждом шаге алгоритма, её размер, конечный результат в отдельный файл 'output_file.txt'. Если таковой уже существует, то он перезаписывается.
